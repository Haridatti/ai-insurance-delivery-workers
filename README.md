# AI-Based Insurance for Delivery Workers

## ❗ Problem Statement
Delivery workers lose income due to external factors like rain, pollution, and curfews. They currently have no financial protection and face income loss.

## 💡 Solution
We propose an AI-powered system that detects external disruptions and automatically provides compensation to delivery workers based on income loss.

## 👥 Users
- Delivery Workers  
- Admin (Insurance System)

## 🔄 Workflow
1. Worker registers  
2. System monitors weather & disruptions  
3. AI detects risk  
4. Automatic claim triggered  
5. Instant payout  

## 🧠 AI Integration
- Predict risk based on weather  
- Detect fraud using location validation  
- Trigger automatic payouts  

## 💰 Weekly Premium Model
- Worker pays small weekly fee (₹50–₹100)  
- Gets protection for income loss  
- Higher risk → higher premium  

## ⚙️ Tech Stack
- Frontend: ReactJS  
- Backend: Node.js  
- Database: MongoDB  
- APIs: Weather API  

## 🔥 Key Features
- Automatic claim system  
- AI-based risk detection  
- Instant payout simulation
